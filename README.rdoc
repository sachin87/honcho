# Honcho

The Simplest, Highly Customizable Administration Framework for Ruby on Rails with Semantic UI.

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'honcho'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install honcho
```
